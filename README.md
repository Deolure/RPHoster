## EN:
RPHoster is a client-side Minecraft mod that allows you to quickly upload your resource packs and share them via a direct link. It supports both folders and ZIP archives and automatically hosts your resource pack using an external file hosting service.

## How to use

1. Place your resource pack in the `resourcepacks` folder.  
2. Open Minecraft and press **T** to open chat.  
3. Use the command: ``/rphost <rp_name>``
4. Wait a few seconds while the mod uploads your resource pack.  
5. You will receive a **download link in chat** — click it to open or copy.

### Requirements

- The resource pack must contain `pack.mcmeta`  
- Supported formats:
  - Folder  
  - `.zip` archive  

## RU:
RPHoster — это клиентский мод для Minecraft, который позволяет загружать ресурспаки и получать на них прямую ссылку. Поддерживает папки и ZIP-архивы, автоматически загружает ресурспак на внешний хостинг.

## Как использовать

1. Помести ресурспак в папку `resourcepacks`.  
2. Запусти Minecraft и открой чат (**T**).  
3. Введи команду: ``/rphost <название_ресурспака>``
4. Подожди несколько секунд — мод загрузит ресурспак.  
5. В чате появится **ссылка на скачивание**, по которой можно перейти.

### Требования

- В ресурспаке должен быть файл `pack.mcmeta`  
- Поддерживаются:
  - папки  
  - `.zip` архивы
